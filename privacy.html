Политика обработки персональных данных

Редакция от 16 июля 2026 года. Документ относится к данным, которые пользователь добровольно передаёт через форму сайта VN-MASTERS.

1. Общие положения

Оператором данных является владелец сайта и технического сервиса VN-MASTERS, расположенного по адресу: Ростов-на-Дону, ул. Стартовая, 1/1. Полные юридические реквизиты оператора подлежат подтверждению собственником. Контакт по вопросам обработки данных: +7 (863) 260-03-45.

Отправляя форму, пользователь подтверждает, что передаёт данные добровольно для связи по своему обращению. Не следует указывать в свободном поле паспортные, платёжные, медицинские и иные сведения, которые не нужны для записи в автосервис.

2. Какие данные могут обрабатываться

  • имя;
  • номер телефона;
  • предпочтительный способ связи;
  • марка, модель и год автомобиля;
  • VIN, если пользователь сам указывает его для подбора запчастей;
  • описание симптома, услуги и истории обращения;
  • предпочтительный день визита;
  • технические данные запроса, которые может обрабатывать поставщик формы для безопасности и доставки сообщения.

3. Цели обработки

  • связаться с пользователем по его заявке;
  • уточнить автомобиль, симптом и направление диагностики;
  • согласовать или подтвердить визит;
  • подготовить предварительную оценку, если для неё достаточно данных;
  • защитить форму от спама и технических злоупотреблений.

4. Способ отправки

Сайт остаётся статическим. Данные формы отправляются через сервис Formspree по существующему адресу формы. Formspree выступает техническим поставщиком доставки сообщения и может обрабатывать данные в соответствии со своими условиями и политикой конфиденциальности. Если пользователь не хочет передавать данные через форму, он может позвонить по одному из телефонов на странице контактов.

5. Основание и срок обработки

Основанием является согласие пользователя, выраженное установкой отметки в форме и её отправкой. Данные должны храниться не дольше, чем это требуется для обработки обращения, выполнения договорённостей и соблюдения обязанностей по закону. Точные внутренние сроки хранения необходимо утвердить у собственника.

6. Передача и защита

Данные не должны публиковаться или передаваться посторонним лицам без законного основания. Доступ получают только лица, которым он нужен для связи и обработки обращения, а также технический поставщик формы. Оператор принимает разумные организационные меры защиты, но передача через интернет не может считаться абсолютно безрисковой.

7. Права пользователя

Пользователь может запросить сведения об обработке, уточнение, прекращение обработки или удаление переданных данных, если их хранение не требуется по закону. Для обращения используйте телефон +7 (863) 260-03-45 и сообщите дату и содержание заявки, достаточные для её поиска.

8. Изменения политики

Актуальная редакция публикуется на этой странице. При изменении формы, поставщика доставки или состава данных политика должна быть обновлена.
