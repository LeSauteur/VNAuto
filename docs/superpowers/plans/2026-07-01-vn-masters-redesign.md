# VN-MASTERS Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Перепаковать статический сайт VN-MASTERS в спокойный премиальный «бутиковый технический сервис», сохранив рабочие контакты, карты, форму и утверждённые скидки 3/5/7/10%.

**Architecture:** Сохраняем текущий многостраничный HTML/CSS/JS без фреймворков. Основная перестройка выполняется в `index.html` и общей системе стилей `assets/css/main.css`; вторичные страницы получают единый тон и безопасные формулировки. Автоматическая проверка на Node.js контролирует обязательные тексты, ссылки, форму и локальные ресурсы.

**Tech Stack:** HTML5, CSS3, vanilla JavaScript, Node.js built-in test runner.
